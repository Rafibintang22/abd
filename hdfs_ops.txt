// ğŸ” Lihat isi direktori (detail size dan waktu)

// ğŸ“„ Melihat isi sebagian file (mirip head)

// âœ… Cek apakah file/direktori ada

// ğŸ” Cek jumlah block & lokasi tiap block

// ğŸ“Š Cek penggunaan storage (disk usage)

// ğŸ“ˆ Ukuran folder total

// ğŸ” Melihat block size dari file

// ğŸ§± Melihat konfigurasi block size default
